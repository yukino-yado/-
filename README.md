# 数学プリント作成ツール v77

## v77の修正内容

- v75ベースです。
- Vercel公開向けに、保存方法を「アプリ内プリント一覧保存」へ変更しました。
- TeXは保存ではなく「出力」として残しています。
- URLを知っていれば誰でも使えるWEBアプリとして運用できます。
- 保存データは各ユーザーのブラウザ内に保存されます。

## 追加したファイルメニュー

- 新規作成
- プリント一覧
- 上書き保存
- 名前を付けて保存
- 複製して保存
- このプリントを削除
- バックアップ出力
- バックアップ読込
- PDF保存 / 印刷
- TeX出力
- 全保存データ退避
- 自動保存を削除

## 使い方

1. プリントを作成します。
2. `ファイル → 名前を付けて保存` を押します。
3. プリント名を入力すると、アプリ内に保存されます。
4. 次回以降は `ファイル → プリント一覧` から開いて再編集できます。
5. 編集後は `ファイル → 上書き保存` で保存します。

## 注意

この方式では、プリントはサーバーではなく各ユーザーのブラウザ内に保存されます。  
別PCへ移す場合は、`バックアップ出力` と `バックアップ読込` を使ってください。

GitHub経由でVercelへ公開する場合は、ZIP内の `index.html`, `script.js`, `style.css`, `README.md` をリポジトリに反映してください。
