# 数学プリント作成ツール v76

## v76の修正内容

- v75ベースです。
- 保存方法をWordに近い流れに変更しました。
- 上部に **ファイル** メニューを追加しました。
- メニュー項目：
  - 新規作成
  - 開く
  - 上書き保存
  - 名前を付けて保存
  - PDF保存 / 印刷
  - TeX出力
  - 全保存データ退避
  - 自動保存を削除
- Chrome / Edge など対応ブラウザでは、`名前を付けて保存` で保存先とファイル名を選べます。
- 一度 `名前を付けて保存` した後は、`上書き保存` で同じファイルに保存できます。
- 対応していないブラウザでは、従来通りダウンロード保存に自動で切り替わります。
- 保存ファイルの拡張子は `.mathprint` です。中身はこれまでのJSON互換です。
- 既存の `.json` ファイルも開けます。

## 注意

ブラウザの安全仕様上、ローカルHTMLアプリではWord完全同等の常時ファイル書き込みはできません。  
ただし、Chrome / Edge の File System Access API に対応している環境では、Wordに近い「名前を付けて保存 → 上書き保存」の流れで使えます。

## 使い方

ZIPを解凍して、`math_print_tool_v76/index.html` をブラウザで開いてください。
